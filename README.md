**Fitur**

- Multi-threaded scanning - Cepat dengan 40 workers 

  - Directory listing (Index of /) 

  - API endpoints (JSON, XML) 

- Dual mode - Normal & Recursive scan 

1. Normal - Scan langsung, tidak masuk ke subdirectory 

	Example : domain.com/admin 

2. Recursive - Masuk ke direktori yang ditemukan (max depth: 3) 

	Example : domain.com/admin/asset 

- False positive filter - Auto skip Cloudflare protection, login pages, 404 errors
